NAME
       ss8_copy_to_cstr,  ss8_copy_to_bytes  −  copy an ssstr byte string to a
       buffer

SYNOPSIS
       #include <ss8str.h>

       bool ss8_copy_to_cstr(ss8str const *restrict str,
                             char *restrict buf, size_t bufsize);
       bool ss8_copy_to_bytes(ss8str const *restrict str,
                             char *restrict buf, size_t bufsize);

DESCRIPTION
       ss8_copy_to_cstr() and ss8_copy_to_bytes() copy  the  contents  of  the
       ss8str  at  str  to the buffer located at buf.  ss8_copy_to_cstr() then
       appends a null byte.

       At most bufsize bytes are written; if bufsize is not sufficient to hold
       the  string  (and  null  terminator for ss8_copy_to_cstr()), the copied
       string is truncated at bufsize ‐ 1 bytes  (for  ss8_copy_to_cstr())  or
       bufsize bytes (for ss8_copy_to_bytes()).

       ss8_copy_to_cstr()  always  places  a null terminator at the end of the
       copied string unless bufsize is zero.

       Behavior is undefined unless str is a valid ss8str object and buf is  a
       buffer  of  at  least  bufsize  bytes  that is not part of the internal
       buffer of str.

RETURN VALUE
       ss8_copy_to_cstr() and ss8_copy_to_bytes() return true  if  bufsize  is
       sufficiently  large  to  copy the whole of the ss8str at str (including
       the null terminator in the  case  of  ss8_copy_to_cstr()).   Otherwise,
       they return false.

SEE ALSO
       ss8_copy_bytes(3), ss8_copy_cstr(3), ssstr(7)
